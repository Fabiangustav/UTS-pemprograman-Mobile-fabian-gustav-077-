. MaterialApp
Fungsi: Widget root yang mengatur tema dan navigasi aplikasi
Properties penting:
theme: Mengatur tema global aplikasi
home: Menentukan halaman pertama yang ditampilkan
2. Scaffold
Fungsi: Struktur layout dasar material design
Properties:
appBar: Header/toolbar aplikasi
body: Konten utama halaman
backgroundColor: Warna latar belakang
3. AppBar
Fungsi: Header/toolbar dengan judul dan styling
Properties modern:
shape: Membuat bentuk rounded corner
elevation: Menghilangkan shadow untuk tampilan flat
4. GridView.builder
Fungsi: Menampilkan data dalam bentuk grid secara efisien
Properties:
gridDelegate: Mengatur layout grid (jumlah kolom, spacing)
itemCount: Jumlah item dalam grid
itemBuilder: Builder untuk setiap item
5. Card
Fungsi: Container dengan elevation dan rounded corners
Properties modern:
elevation: Shadow effect
shape: Rounded corners dengan BorderRadius
6. InkWell
Fungsi: Memberikan efek ripple ketika ditekan
Properties:
onTap: Aksi ketika ditekan
borderRadius: Sesuai dengan parent Card
7. Container dengan Gradient
Fungsi: Background gradient untuk efek visual modern
Properties:
decoration: Gradient dan borderRadius
8. Column & Row
Fungsi: Mengatur layout vertikal dan horizontal
Properties:
mainAxisAlignment: Penyelarasan utama
crossAxisAlignment: Penyelarasan silang
9. Icon & Text
Fungsi: Menampilkan ikon dan teks dengan styling
Properties modern:
Warna dinamis berdasarkan kategori
Font weight dan size yang konsisten
